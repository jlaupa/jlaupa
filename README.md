### About me

I've spent **11+ years** building backend and APIs, from Peru to Argentina to Spain. Today I'm **Tech Lead for the AI APIs at Magnific (Freepik)**, and I own the API product end to end: architecture and engineering, plus pricing, monetization and roadmap. I built **Magnific's first MCP server** and led the migration from pay as you go to a **usage based credits** system. Over one year the API product grew its revenue about **5x**.

I work **AI first**: I run a multi agent setup with **Claude Code**, one per repository, and I treat LLMs, **MCP** and agents as real accelerators. I care about clean API contracts (**OpenAPI**), reliable gateways, and billing that adds up. I'm a hands on **individual contributor** at heart: I want to keep shipping product with AI, close to the code.
